A simple and intelligent Python-based Bio Agent that interacts with the user to collect personal details such as name, date of birth, email, phone, and address. It also cleans, extracts, and formats the user’s answers—ensuring only the main information is stored. Users can review and edit any field before finalizing.

Features:

-> LLM-powered extraction : Automatically extracts clean values from natural sentences (e.g., “my email is abc@gmail.com” → abc@gmail.com).

-> Smart Address Cleaning : Keeps only the main location and pincode (e.g., “I live in Bangalore 560010” → Bangalore 560010).

-> Interactive Editing : Users can modify any field: Name, DOB, Email, Phone, Address.

-> Final Summary Output : Prints all details in a clean, professional format—perfect for resume pipelines.

-> Works with API keys : Supports OpenAI or Gemini LLMs.
